# Supli AI — React Native App

App mobile de suplementação personalizada para o mercado brasileiro.

**Site:** https://supliai.com  
**Bundle ID:** com.supliai  
**Plataformas:** iOS + Android

## Stack
- React Native (Expo)
- TypeScript
- NativeWind (Tailwind)

## Telas
1. Home — checklist de suplementos do dia
2. Progresso — streak, biomarcadores, conquistas
3. Bem-estar — mascote reativo, hábitos, biohacking
4. Perfil — plano, configurações
