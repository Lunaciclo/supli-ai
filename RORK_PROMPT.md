# Rork Prompt — Supli AI

Use este prompt no Rork para gerar o app:

---

Create a React Native mobile app called "Supli AI" for iOS and Android.

The app already exists as a PWA at https://supliai.com — use it as visual and functional reference.

Bundle ID: com.supliai
App name: Supli AI
Primary color: #2D6A4F (forest green)
Background: #F5F2EC (warm cream)
Accent: #74C69D
Font: System font (similar to Inter)
Mascot: capsule pill character (purple-to-green gradient with eyes)

The app has 4 screens (bottom navigation):

1. HOME (🏠)
- Personalized supplement checklist for the day
- Each supplement shows: name, dose, time, check button
- Rotating mascot tips at top
- Daily habits section (water, sleep, exercise)
- Current streak display

2. PROGRESSO (📈)
- Streak hero card (days in a row)
- Biomarker evolution charts (Vitamina D, Ferritina, B12, Magnésio)
- Achievements/badges grid
- Health scores over time

3. BEM-ESTAR (🌿)
- Reactive mascot (changes expression based on habits logged)
- Biohacking tips of the day
- Habit logging buttons
- Smart reminders section

4. PERFIL (👤)
- User name and plan badge
- Protocol countdown (90 days)
- Settings
- Logout

Onboarding flow (before main app):
- Splash screen with animated mascot
- Quiz (5 questions: goals, symptoms, diet, sleep, activity level)
- Loading screen "Gerando seu protocolo..."
- Protocol reveal screen

The app should feel premium, clean, health-focused. Brazilian Portuguese language throughout. No barcode scanner needed.
